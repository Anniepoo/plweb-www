---+ SWI-Prolog Support

If you have problems with SWI-Prolog there are a number of places you
can go for help from the community:

    * The [[FAQ][</FAQ/>]].

    * The Mailinglist.txt and its archive.  Tip: include =SWIPL= to
    search the mailinglist archives.

    * If you encounter a bug.txt, please report it using [[our issue
    tracker][https://github.com/SWI-Prolog/swipl-devel/issues]].
    Notice that each package has a separate issues tracker, e.g., the
    [[issue tracker of the HTTP
    package][https://github.com/SWI-Prolog/packages-http/issues]], or
    the [[issue tracker of the SSL
    package][https://github.com/SWI-Prolog/packages-ssl/issues]].

    * For Debian-specific problems there is [[a Debian bug
    tracker][http://bugs.debian.org/cgi-bin/pkgreport.cgi?pkg=swi-prolog]]
    which informs the Debian maintainer Michael Piefel.

    * Join the Prolog IRC channel ##prolog on FreeNode.

    * Check the Newsgroup
    [[comp.lang.prolog][<news:comp.lang.prolog>]]

    * Search for [[Prolog-tagged questions on
    StackOverflow][https://stackoverflow.com/questions/tagged/prolog]]
