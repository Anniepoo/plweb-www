---++ Status

SWI-Prolog is available in Mageia 3 repositories. The packages are:

* =swi-prolog= - Prolog interpreter and compiler
* =swi-prolog-jpl= - Java interface for swi-prolog
* =swi-prolog-xpce= - swi-prolog native GUI library
